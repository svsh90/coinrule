## What Is Coinrule, Anyway?

Coinrule is a no-code automated trading bot platform founded in London in 2018. The idea is simple: you tell it *when* to buy or sell, and it goes and does it — 24 hours a day, across whatever exchange you're already using.

No programming. No terminal. No PhD required.

You build rules using what they call "If-This-Then-That" logic. Something like: *"If BTC drops 5% in 4 hours, buy $200 worth."* Or: *"If ETH rises 8% with volume up 20%, take 30% profit."* You set the logic, launch the bot, and go live your life.

## Is Coinrule Safe?

Non-custodial is the key word here. Coinrule never holds your funds. It connects to your exchange via API keys that are set with no withdrawal permissions — the bot can trade but can't move money off the exchange. Your crypto stays where it is.

API keys are encrypted with AES-256, which is the same standard banks use. The platform itself has been operating since 2018 without any reported major security incident.

One thing to be clear about: Coinrule isn't a get-rich scheme. Whether your bots make money depends on whether your strategies are any good — the platform executes faithfully, but it can't guarantee profitable outcomes. Use the demo exchange to validate before going live with real capital.
